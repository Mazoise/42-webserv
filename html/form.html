Send data ?

We use, in majority, HTML form to send data in a http context. A form points to a ressource (a PHP file for example) on the server with a script for processing input data. This target is call action in a form.

A form also needs to specify which method will be used, GET or POST, to send the request.

<form method="GET" action="form_get.php">
. . .
</form> HTML

prerequisite : the targets of this forms are PHP files, the server must be correctly setup to run this CGI.

Method GET

The GET method send data of the form directly in URI.
Try to send data with GET, click on submit button.

Method POST

The POST method send data of the form in body request.
Try to send data with POST, click on submit button.
